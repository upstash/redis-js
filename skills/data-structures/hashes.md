# Hash Operations

## What This File Covers

- HSET, HGET, HMGET, HGETALL with object serialization
- HINCRBY, HINCRBYFLOAT for numeric fields
- HDEL, HEXISTS operations
- HKEYS, HVALS, HLEN for inspection
- Storing JavaScript objects as hashes
- When to use hashes vs JSON strings
- Common patterns: user profiles, configuration, structured data

## Key Topics

1. **Basic Hash Operations**: HSET/HGET with automatic object handling
2. **Retrieving Data**: HGETALL returns proper objects
3. **Numeric Fields**: HINCRBY with real numbers
4. **Batch Operations**: HMGET, HMSET
5. **Use Cases**: User data, settings, entity storage
6. **Hashes vs JSON**: When to use each approach
