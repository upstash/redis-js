# JSON Operations

## What This File Covers

- JSON.SET, JSON.GET for storing and retrieving JSON documents
- JSONPath queries for accessing nested fields
- JSON.ARRAPPEND, JSON.ARRINDEX for array operations
- JSON.NUMINCRBY for numeric operations on nested fields
- JSON.DEL for deleting paths
- When to use JSON vs regular serialization
- Complex nested object storage
- Common patterns: document storage, configuration, complex entities

## Key Topics

1. **Basic JSON Operations**: JSON.SET/GET with nested objects
2. **JSONPath**: Querying nested fields with $ syntax
3. **Array Operations**: Manipulating arrays within JSON documents
4. **Numeric Operations**: NUMINCRBY on nested numbers
5. **Partial Updates**: Updating specific paths without full replacement
6. **Use Cases**: User profiles, product catalogs, configuration
7. **JSON vs Standard Serialization**: When to use which approach
