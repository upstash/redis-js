# Stream Operations

## What This File Covers

- XADD for adding entries to streams
- XREAD, XREADGROUP for consuming streams
- XGROUP for consumer group management
- XACK for acknowledging messages
- XLEN, XRANGE for stream inspection
- Building event streams and message queues
- Consumer groups for distributed processing
- Common patterns: event sourcing, activity logs, message queues

## Key Topics

1. **Adding Entries**: XADD with automatic ID generation
2. **Reading Streams**: XREAD with blocking and non-blocking modes
3. **Consumer Groups**: XGROUP for distributed consumption
4. **Acknowledgment**: XACK for reliable processing
5. **Stream Management**: XTRIM, XDEL for cleanup
6. **Use Cases**: Event logs, chat systems, real-time feeds
7. **Patterns**: Fan-out, work queues, event sourcing
