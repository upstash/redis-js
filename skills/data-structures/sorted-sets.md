# Sorted Set Operations

## What This File Covers

- ZADD, ZREM, ZRANGE, ZREVRANGE operations
- ZSCORE, ZRANK, ZREVRANK for scoring and ranking
- ZINCRBY for updating scores
- ZRANGEBYSCORE, ZCOUNT for range queries
- Building leaderboards and rankings
- Time-based sorting (using timestamps as scores)
- Common patterns: leaderboards, priority queues, trending content

## Key Topics

1. **Basic Sorted Set Operations**: ZADD with scores and members
2. **Range Queries**: ZRANGE, ZREVRANGE with WITHSCORES
3. **Scoring**: ZSCORE, ZINCRBY for score management
4. **Ranking**: ZRANK for position lookup
5. **Leaderboards**: Complete leaderboard implementation
6. **Time-Series**: Using timestamps for time-based sorting
7. **Use Cases**: Game leaderboards, trending items, priority queues
